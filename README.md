## Tasks

### Task 1: Static Webpage

- **Description**: A simple static webpage to demonstrate basic HTML and CSS layout skills.

![Static Webpage](https://img.shields.io/badge/Status-Done-green)

### Task 2: React Dashboard

- **Description**: A modern React-based dashboard featuring:
  - Modular layout (Header, Sidebar, Notification Dropdown)
  - Multiple pages (Overview, Profile, Projects, Statistics, Messages, Download Report, Edit Profile)
  - **Tailwind CSS** for styling
  - **Responsive** and clean UI
  - **React Router** for navigation
- **Features**:
  - User-friendly navigation
  - Dynamic data handling

![React Dashboard](https://img.shields.io/badge/Status-In%20Progress-yellow)

### Task 3: Weather Dashboard

- **Description**: An interactive weather dashboard built with:
  - **OpenWeatherMap API** for real-time weather data
  - **Leaflet.js** for a map preview with weather overlays (temperature, wind, precipitation, clouds, etc.)
  - 3-day weather forecast
  - **Dark Mode Toggle**
  - Search and location pin functionality
- **Technologies**:
  - **JavaScript**
  - **CSS** for responsive design
  - **Weather API Integration**

![Weather Dashboard](https://img.shields.io/badge/Status-Done-green)

### Task 4: Personal Blog Homepage

- **Description**: A React-based personal blog homepage with:
  - **Card layout** for blog posts
  - Filtering and sorting features
  - **Responsive design** for mobile and desktop
- **Technologies**:
  - React
  - Tailwind CSS

![Blog Homepage](https://img.shields.io/badge/Status-Done-green)

## How to Run

### For Static Tasks (1 & 3):
1. Download the repository.
2. Open `index.html` directly in your browser.

### For React Tasks (2 & 4):
1. Clone the repository.
2. Navigate to the folder: `task2/` or `task4/`.
3. Run `npm install` to install dependencies.
4. Run `npm start` to start the development server.
5. Open the application in your browser at `http://localhost:3000`.
